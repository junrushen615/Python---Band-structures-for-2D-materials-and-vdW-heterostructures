# 2D Band Structure Visualization Toolset
This repository provides a streamlined Python pipeline for the generation and visualization of 2D electronic band structures. It includes optimized `.xyz` coordinates and plotting `.py` scripts specifically tailored for outputs from the [TURBOMOLE RIPER](https://www.turbomole.org/) module and the [streamlit](https://riper-tools.streamlit.app/) app introduced by Dr. Manas Sharma.

The data is organized by material type to ensure scalability and ease of access.
